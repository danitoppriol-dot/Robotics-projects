# Robotics Projects

This repository contains the assignments and final project developed for a Robotics course.

## Repository Structure

The repository is organized into independent modules, one for each assignment and the final project.

assignment_1_kinematics/  
Inverse kinematics for SCARA and KUKA manipulators.

assignment_2_path_planning/  
Search-based motion planning for a Dubins car model.

assignment_3_mapping/  
Occupancy grid mapping and configuration space inflation.

final_project/  
Behavior Tree implementation for autonomous robot control.


## Technologies

- Python 3  
- NumPy  
- ROS2 (for mapping and final project components)
